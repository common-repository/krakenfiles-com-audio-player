Replace all KrakenFiles links to embed audio player

== Description ==

This plugin converts all KrakenFiles links in your Wordpress content into Krakenfiles audio player.

== Installation ==

1. Upload plugin krakenfiles-audio-player.zip
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to plugin settings page and choose your width and download button place

== Frequently Asked Questions ==

= How can I change the download button? =

Just put your `download_button.png` to the `/wp-content/plugins/krakenfiles-audio-player/images` folder and replace it
